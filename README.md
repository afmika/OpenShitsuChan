# OpenShitsuChan
OpenSource version of Shitsu-chan (https://shitsulab.herokuapp.com)

# How to start
1. Create a database
2. Load the file stored in /assets/dump in your local postgres database
3. Configure project.config.json
4. Type `node server.js`
5. .. and you are done !
